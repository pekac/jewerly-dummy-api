# Jewerly webshop API

Small NestJS API made for react jewerly shop:
https://github.com/andjared/typescript_react_webshop

## Installation

Make sure to first install docker on your machine.

```bash
$ npm install
```

## Running the app locally

```bash
# start db first
$ docker-compose up -d
# start api next
$ npm run start:dev
```
